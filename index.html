Run

From canon to artifact

  • Start Seed: ./scripts/seed.sh start (API 8123, Observer 9000).
  • POST /event or /trace to log missions and worker activity.
  • GET /log/stream to watch canon move toward artifacts.
  • View docs and status without leaving the runtime.

OEP

Epistemics with guardrails

Envelope keeps obligations/refusals/unknowns explicit.

obligations: cite_sources, mark_hypotheses
refusals: missing_provenance, user_constraints_violation
epistemic_types: claim, hypothesis, observation, refusal
envelope: sha256 evidence hash, lineage: genesis_key_fingerprint

Identity

Deterministic lineage

Genesis Key fingerprints and Symbia Key constraints show up in status, identity snapshots, and traces. Evidence stays probabilistic; lineage stays deterministic.
